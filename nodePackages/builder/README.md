# 🚀 Builder for LaunchPad Icons

LIB: _LunchPad Icons Builder_

## Usage

```sh
pnpm run build
```

```sh
lib-codeBuilder react /path/to/iconsPath > /path/to/react.tsx
```

```sh
lib-codeBuilder solid-js /path/to/iconsPath > /path/to/solid.tsx
```
